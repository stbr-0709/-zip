# 🎭 KALEIDO - 万花筒男团叙事游戏

## 游戏简介

> “你所看到的，只是万花筒的一面。”

KALEIDO是一款**角色扮演+叙事选择**的网页游戏。你将扮演第六名成员，加入一个全员“表里不一”的男团，在谎言与真相之间做出选择，走向不同的结局。
